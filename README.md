# Auto Spotter


💻 Auto Spotter is a work in progress personal project that is aimed at using CoreML and Swift in tandem to help with the car spotting hobby. 

🚗 Users can effortlessly keep track of cars they spot through an easy to navigate app.

📊 This app uses The Comprehensive Cars dataset, which was converted to CoreML model using coremltools python package. This can help identify over 431 different car models.

🔥 Firebase is used for user authentication allowing users to store the info of their favourite cars they spot to their accounts using Firestore.
